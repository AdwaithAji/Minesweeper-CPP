# Minesweeper Game

##How to Play
1.Choose difficulty : (easy(2 bombs), medium(4 bombs), hard(6))
2.Enter position    : Enter a two-digit number to pick a spot.
3.Win condition     : Clear all safe spot to win the game.
3.Exit conditoin    : Type "exit" to quit the game.

##How to Run
Compile the code using any C++ compiler (like G++):
`g++ main.cpp -o minesweeper`
`./minesweeper`
